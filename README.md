# solidity-test-contract2

This repository is a minimal starter for an "external PR" exercise.

Goal:
- Allow external contributors to fork this repository, make a small change (e.g. improve the contract or add a README note), and open a Pull Request against the `main` branch.

Starter files:
- contracts/SimpleGreeting.sol â€” a minimal Solidity contract that stores and returns a greeting

How to contribute (short):
1. Fork this repo to your GitHub account.
2. Clone your fork locally.
3. Create a new branch for your change: `git checkout -b my-fix`
4. Make your changes, stage and commit them.
5. Push the branch to your fork and open a PR from `your-branch` -> `JohnmcKennith:main`.

Example of a small change:
- Update the contract constructor default greeting or add a line in the README.
